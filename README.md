## Projects

### 01 · IvoryFits — Luxury Fashion E-Commerce

A full-stack luxury fashion e-commerce platform with a premium frontend experience, AI-powered stylist assistant, and a complete admin dashboard for product and order management.

**Live:** https://ivoryfits.onrender.com

**Stack:** HTML · CSS · JavaScript · Node.js · Express.js · MySQL

**Key Features**
- Product catalog with category, price, and sort filters
- Real-time search overlay and cart with localStorage persistence
- Multi-step checkout with order confirmation flow
- AI personal stylist chatbot
- Admin dashboard — CRUD operations on products and orders
- Light/Dark mode · Fully responsive

### 02 · BuzzBee — Music Social Platform

A real-time music social platform connecting artists, creators, and listeners. Features a live social feed, AI music assistant, story system, and trending music discovery.

**Live:** https://buzz-bee-five.vercel.app

**Stack:** HTML · CSS · JavaScript · Node.js · Express.js · MongoDB Atlas · Socket.io · Cloudinary · Groq (LLaMA 3.3-70b)

**Key Features**
- Social feed with posts, reactions, and threaded comments
- Real-time messaging and notifications via Socket.io
- 24-hour stories with photo/video support
- Vibey AI — music assistant powered by Groq LLaMA 3.3-70b
- Trending music discovery via Jamendo API
- Custom avatar builder · Dark/Light mode

### 03 · WorkNest — Project Management Platform

An AI-powered project management platform built with a modern React/TypeScript frontend and a RESTful Node.js backend, designed for team collaboration and productivity.

**Live:** https://worknest12.netlify.app

**Stack:** React · TypeScript · Vite · Tailwind CSS · Node.js · Express.js · MySQL · Netlify · Render

**Key Features**
- JWT-based authentication with secure session management
- Workspaces and multi-project organization
- Kanban boards with drag-and-drop task management
- Team collaboration with role-based invitations
- Activity logs, notifications, and real-time updates
- Gamification — XP system, levels, and streaks
